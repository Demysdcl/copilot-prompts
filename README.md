# Project Documentation & AI Assistant Guidelines

This folder contains documentation detailing the different AI Copilot modes available to assist with the development of the **Dance School Empire** application.

These documents explain the distinct objectives, capabilities, and best ways to interact with each AI mode, ensuring they are utilized effectively alongside our specific technology stack (React Router v7, Prisma, Tailwind CSS, etc.).

## AI Interaction Modes

The AI assistance is divided into three distinct modes, tailored for different stages of your development workflow:

*   **[Plan Mode](./plan.md)**: Your strategic architect. Use this mode to design features, plan database schemas, and outline application architecture before implementation begins. It does not write code.
*   **[Ask Mode](./ask.md)**: Your read-only repository expert. Use this mode to ask questions, understand existing code, learn about library patterns, or debug issues without making file changes.
*   **[Agent Mode](./agent.md)**: Your autonomous implementer. Use this mode when you have a clear plan or task and need the AI to actively write code, run terminal commands, and build out the feature end-to-end.

Please review the individual markdown files to learn how to frame prompts and tasks perfectly for each specific mode.
